## Sequence of Instructions (Do not deviate, skip, or optimize)

### 1. Calcular Variáveis Automáticas

**Obter data atual do sistema:**
- Extract current year (4 digits) → `current_year_full`
- Calculate `current_year` = current_year_full - 2000 (2 digits)
- Extract current month → `current_month` (without leading zero: 1-12)

**Calcular valores:**
```
COMPOSE_PROJECT_NAME = {io_project}_{io_app}_development
IO_VERSION = 0.{current_year}.{current_month}-dev.1
```

**Exemplos de IO_VERSION:**
- Outubro/2025 → `0.25.10-dev.1`
- Julho/2025 → `0.25.7-dev.1`
- Janeiro/2026 → `0.26.1-dev.1`

### 2. Gerar Conteúdo do .env.io

Generate content:
```bash
# Embrapa I/O Environment Variables
# Generated by BMAD Embrapa I/O Module
# This file should NOT be committed to version control
# Use .env.io.example as reference

# ==========================================
# Docker Compose Project Configuration
# ==========================================
COMPOSE_PROJECT_NAME={io_project}_{io_app}_development
COMPOSE_PROFILES=development

# ==========================================
# Embrapa I/O Platform Variables
# ==========================================
IO_SERVER=localhost
IO_PROJECT={io_project}
IO_APP={io_app}
IO_STAGE=development
IO_VERSION=0.{current_year}.{current_month}-dev.1
IO_DEPLOYER={io_deployer}

# ==========================================
# Observability - Sentry & Matomo
# ==========================================
# Get your SENTRY_DSN from https://dashboard.embrapa.io
SENTRY_DSN=GET_IN_DASHBOARD
MATOMO_ID=522
MATOMO_TOKEN=
```

Store as `{env_io_content}`.

### 3. Gerar Conteúdo do .env.io.example

Generate content (same as .env.io but with sanitized deployer):
```bash
# Embrapa I/O Environment Variables - EXAMPLE
# Copy this file to .env.io and fill in your values
# DO NOT commit .env.io to version control

# ==========================================
# Docker Compose Project Configuration
# ==========================================
COMPOSE_PROJECT_NAME={io_project}_{io_app}_development
COMPOSE_PROFILES=development

# ==========================================
# Embrapa I/O Platform Variables
# ==========================================
IO_SERVER=localhost
IO_PROJECT={io_project}
IO_APP={io_app}
IO_STAGE=development
IO_VERSION=0.{current_year}.{current_month}-dev.1
IO_DEPLOYER=your.email@embrapa.br

# ==========================================
# Observability - Sentry & Matomo
# ==========================================
# Get your SENTRY_DSN from https://dashboard.embrapa.io
SENTRY_DSN=GET_IN_DASHBOARD
MATOMO_ID=522
MATOMO_TOKEN=
```

Store as `{env_io_example_content}`.
